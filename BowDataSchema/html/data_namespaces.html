DB Namespaces

  • DB 0 - Basement
    • State flags.
    • Primitive templates.
    • Test drivers.
    • Experimental and prototype structures.
    • Use DB Editor to explore, edit Basement DB.
  • DB 1 - Schema
    • Message schema metadata. Record templates.
    • For example, topic and audit templates.
    • Use DB Editor to explore, edit Schema DB.
  • DB 2 - Harvest
    • Message response packets.
    • The messaging gateway writes response content to the Harvest DB.
    • Receivers pull content using a token = a Redis unique key to Harvest DB.
    • Use Monitor to explore Harvest DB.
  • DB 3 - Log
    • Detailed messaging system log.
    • The messaging system Wire Tap writes forensic info to Log DB.
    • Use Monitor to explore Log DB.
  • DB 4 - Monitor
    • Analytical messaging system log and summarizer.
    • The messaging system Wire Tap writes summary and state info to Monitor DB.
    • Use Monitor to explore Monitor DB.
