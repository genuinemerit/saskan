Services Architecture

See the application architecture page for an overview of what a service-oriented application architecture does.

The message-based services rely primarily on the Python Asyncio library. They are designed to be modular and extensible.

Main categories of software components that manage the services:

  • Services Metadata
  • Service Control
  • Service Request
  • Service Response
  • Service Payload construction and sequencing
  • Activation of services
  • Monitoring of services
