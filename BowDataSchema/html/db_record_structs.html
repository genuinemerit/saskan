Record Structures

Records of the same type have the same structural pattern:

  • A Key Format is the same for all records of the same type. Like...
    • Category:Name:Version
      • Example: Redis:NamespacesSchema:0.1.0
    • TopicName:MessageName:MessageAction:Version
      • Example: Ontology:GetOntologyFile:request:0.1.0
      • Example: Ontology:GetOntologyFile:response:0.1.0
  • The Values Format identifies fields to be passed in on a request or to be provided as a result on response or publication.
  • The Audit Format is the same for all records.
    • HashID: hash_value
    • LastModified: timestamp+sequence_number
